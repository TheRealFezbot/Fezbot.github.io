Fezbot's HTML Quick Reference Sheet

HTML tags

Tag Name Description
<html> HTML Tag Contains all HTML elements
<head> header tag Contains metadata and links to CSS en scripts needed for the webpage
<body> body tag Contains the content of the webpage
<link> link tag link to CSS files
<meta> meta tag Contains metadata
<title> title tag Pagetitle in the browser
<script> script tag links scripts or defines scripts, loads for <body> content
<a> hyperlink tag Encases a hyperlink
<button> button tag Creates a button

Table Tags

Tag Name Description
<table> Table The wrapper of elements for all HTML tables
<thead> Table head The set of rows defining the colum headers in a table
<tbody> Table body The set of rows containing actual table data
<tr> Table row The table row container
<tr> Table row The table row container
<td> Table data The table data container
<tfoot> Table foot The set of rows defining the footer in a table
